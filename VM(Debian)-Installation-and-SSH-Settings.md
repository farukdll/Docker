#### VM(Debian)-Installation and Settings
  <h1 align="center">
  <p>
      <img height="350" width="700" src="https://github.com/farukdll/Docker/assets/97880185/b184eaf2-7652-451f-a638-12d5ffcc1fca">
  </h1> <p> </p>

  * ###### VirtualBox official website [⇨ Click-Here ⇦](https://www.virtualbox.org/wiki/Downloads)
  * ###### First of all, enter the VirtualBox official web page via the link above and download VirtualBox according to the operating system of the computer you are using.
  * ###### Then install VirtualBox on your computer.
  
  <h1 align="center">
  <p>
      <img height="220" width="700" src="https://github.com/farukdll/Docker/assets/97880185/11c49430-0489-4125-a3cb-7e3330f222c5">
  </h1> <p> </p>

  * ###### Debian official website [⇨ Click-Here ⇦](https://www.debian.org/download)
  * ###### Then go to Debian`s official website via the link above and download the latest version ISO file.

  <h1 align="center">
  <p>
      <img height="300" width="700" src="https://github.com/farukdll/Docker/assets/97880185/c24b6812-1be4-41b4-a6dd-73ff6f4f1b3d">

  * ###### After the VirtualBox installation process is finished, open VirtualBox and click on the "New" tab indicated by the red arrow.
  * ###### When you type debian in the "Name:" field in Image 1 below, the bottom lines will be set automatically (as shown in Image 1: "Type and Version").
  * ###### Then adjust the 4 images below in order according to your computer specifications (as desired).

  <p>
      <img height="270" width="450" src="https://github.com/farukdll/Docker/assets/97880185/610254f7-80f5-42ad-b338-0e3040ca9d02">

  </h1> <p> </p>

  * ###### "General→Advanced→Shared Clipboard" tab, select the 2 options "Bidirectional".
    * ###### The reason for choosing "Bidirectional" is that you can easily transfer (drag and drop) files from the Desktop to the Debian interface.
  * ###### Then set the image memory in the "Display→Image Memory" tab.
    * ###### On the Display tab, check "Enable 3D Acceleration".
    * ###### Used to improve 3D graphics performance. This option is a feature provided by the virtual machine manager (such as VMware, VirtualBox or Hyper-V).
  
  <h1 align="center">
  <p>
      <img height="270" width="450" src="https://github.com/farukdll/Docker/assets/97880185/4071c53c-6415-44de-96a7-eaeb5a363482">

  </h1> <p> </p>
  
  * ###### Go to the "Storage" tab and click on the free disk space indicated by the left red arrow in image 1 and then click on the disk indicated by the right red arrow.
    * ###### In the window that opens, select the Debian-iso file you downloaded.
    * ###### After selecting the Debian-iso file, it should be like the area indicated by the red arrow in the 2nd image.
  * ###### Click "OK" after performing the above steps in order.
